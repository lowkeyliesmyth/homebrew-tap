# Lowkeyliesmyth Tap

## How do I install these formulae?

`brew install lowkeyliesmyth/tap/<formula>`

Or `brew tap lowkeyliesmyth/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "lowkeyliesmyth/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
